HTML means Hyper Text Markup Language HTML is used to create web pages HTML is the backbone of a website HTML elements are represented by tags HTML Works with elements while CSS works with properties CSS stands for Cascading Style Sheets CSS is used to control the layout and appearance of a website CSS is used to separate the presentation of a document from its structure HTML is literally the structure of web pages HTML elements are divided into semantic and non semantic elements Semantic elements are used to define the meaning of the content Non-semantic elements are used to define the layout of the content Examples of semantic elements are: * header * nav * main * section * article * aside * footer Examples of non semantic elements are: * div * span Div is a block element Span is an inline element
